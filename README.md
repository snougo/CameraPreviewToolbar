# Godot Camera Preview Toolbar

一个简单的 Godot 4 编辑器插件，旨在优化 3D 场景的相机预览工作流。

它在 3D 编辑器顶部的工具栏中添加了一个**常驻的“预览相机”按钮**。即使你当前选中了其他节点（如灯光、模型等），只需点击该按钮即可一键预览相机视角，再次点击即可退出并**自动恢复**之前的选中状态。

## 🎯 痛点与解决方案

**痛点：**
在原生 Godot 编辑器中，想要预览 Camera3D 的视角，你必须：
1. 在场景树（Scene Tree）中翻找并选中 `Camera3D` 节点。
2. 在视图左上角勾选 "Preview" 复选框。
3. 预览完后，取消勾选，再重新去寻找并选中你刚才正在编辑的模型。

**解决方案：**
本插件将这一繁琐过程简化为**一次点击**：
- 🖱️ **一键预览**：无论当前选中什么，点击工具栏按钮直接进入相机视角。
- 🔍 **自动寻找**：自动查找场景中的主相机。
- ↩️ **自动恢复**：退出预览时，自动重新选中你之前正在编辑的节点，无需手动切回。

## ✨ 特性

- **无缝集成**：按钮位于 3D 编辑器原生菜单栏中，使用 Godot 原生图标，外观协调。
- **智能显隐**：自动处理原生 "Preview" 复选框的状态。
- **上下文记忆**：记录并恢复预览前的节点选区（Editor Selection）。
- **零配置**：安装并启用后即可直接使用。

## 📦 安装方法

1. **下载源码**：下载本仓库的 ZIP 文件或通过 Git 克隆。
2. **放入项目**：将 `addons/camera_preview_toolbar` 文件夹复制到你的 Godot 项目的 `res://addons/` 目录下。
3. **启用插件**：
    - 打开 Godot 编辑器。
    - 进入 `Project` -> `Project Settings` -> `Plugins`。
    - 找到 "Camera Preview Toolbar" 并将 Status 勾选为 **Enable**。

## 🛠️ 使用说明

1. 确保你的 3D 场景中至少包含一个 `Camera3D` 节点。
2. 在 3D 视图顶部的工具栏（通常在 View 菜单附近），你会看到一个新的相机图标按钮。
3. **点击按钮**：进入相机预览模式。
4. **再次点击**：退出预览模式，并继续编辑之前的对象。
